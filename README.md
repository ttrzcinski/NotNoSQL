# NotNoSQL
Library to handle slice SQL Queries down to run them on NoSQL engine.

Usage:

SQL QUERY --> [ NotNoSQL ] --> [ NoSQL ]

RESULT    <-- [ NotNoSQL ] <-- [ NoSQL ]
